Artigos Científicos

SmartHome - Automação Residencial controlada por página web

Nesse artigo científico é proposto um sistema de baixo custo com uma 'interface' da página web com fácil utilização, o mesmo trabalha com sensores de controle ligados a um Arduíno para o controle de equipamentos domésticos como lâmpadas e janelas. Espera-se com esse projeto proporcionar mais conforto para o morador e também promover para o mesmo uma maior segurança e diminuição de seus gastos.

ARDUINO: UMA TECNOLOGIA NO ENSINO DE FÍSICA

Nesse artigo científico é mostrado o Arduino como uma plataforma alternativa na aquisição automática de dados em experimentos didáticos de Física via porta USB do computador, desse modo sendo desenvolvidos experimentos didáticos que permitam um ensino e uma aprendizagem de Física mais significativo.

A UTILIZAÇÃO DO EMULADOR DE ARDUINO TINKERCAD PARA O ENSINO DE LÓGICA DE PROGRAMAÇÃO E ELETRÔNICA

Nesse artigo científico é utilizado o simulador de circuitos Tinkercad como forma de suprir o Arduino físico, pois oferece os principais componentes eletrônicos e o ambiente de programação totalmente online e gratuito. O objetivo da pesquisa foi de identificar os principais componentes disponíveis na plataforma Tinkercad, apresentar experimentos e a utilização da Linguagem Programação C através do Arduino.
